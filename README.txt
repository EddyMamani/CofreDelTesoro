Nombre de la Aplicación:  Cofre del Tesoro

Esta aplicación expone servicios API REST para que se pueda simular el flujo de efectivo tomando en cuenta las restricciones para los casos de crearCuenta, deposito y retiro

con métodos:

/cofretesoro/crearCuenta
/cofretesoro/depositarDinero
/cofretesoro/retirarDinero
/cofretesoro/saldo
/cofretesoro/historico


Tecnologías/framework usadas:

Spring boot 2.6.0  web, jpa
sqlite
Maven 3.8.4
JDK 8


ejecutar la aplicación.

java -jar cofreDelTesoro-1.jar
